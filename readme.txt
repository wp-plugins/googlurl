Use Google's URL Shortener Goo.gl to create, track or analyze your custom URL. 

== Description ==
Bring the power and reliable URL shortner from Goo.gl to your dashboard. 
This plugin will let you to create shortened URL from administrator dashboard. 
Each shortened URL will be updated daily and you can analyze it by total clicks, referrer sites, 
user browsers, operating system platforms and user countries.

<h4>Key Features & Options</h4>
<ul>
<li>Create as many shortened URL from your dashboard.</li>
<li>No authentication needed.</li>
<li>Analytic reports including total clicks, referrers, browsers, platforms and countries.</li>
<li>Updates daily, or use your own schedule.</li>
<li>Google graphs visualization.</li>
</ul>


== Installation ==
You can use the built in installer and upgrader, or you can install the plugin manually.
<ul>
<li>Go to the menu 'Plugins' -> 'Install' and search for 'Googlurl'</li>
<li>Click 'install'</li>
</ul>
